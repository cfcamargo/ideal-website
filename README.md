# Ideal website

- Esse projeto consiste em um Site para a Empresa Ideal Tecnologia.
- A empresa presta servicos de assistencia técnica agrícola

## Tecnologias Utilizadas
- HTML
- CSS
- Grid Layout
- Flex Box
- Java Script
- Biblioteca UIkit
- Biblioteca Form Submit
- Biblioteca Scroll Reveal
